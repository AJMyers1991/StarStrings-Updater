# StarStrings-Updater

StarStrings-Updater is an auto-install/update utility for the StarStrings modification of Star Citizen made by MrKraken (link to his repository at the bottom of readme).  

This utility installs an executable into C:\Program Files.  This executable allows you to install and update StarStrings across multiple installations of Star Citizen as well as automate the updates so that you will always have the most recent version without needing to lift a finger.  This is also a properly registered program so uninstallation is as easy as removing any other program from either the control panel or system settings.  Logging and settings files can be found in %appdata%\local\StarStrings-Updater.  Scheduled update tasks can be viewed or modified in Windows Task Scheduler under the same name.

If you have v1 of my updater script and/or update-checker script, you can delete those batch files and delete the scheduled task from your Windows Task Scheduler before installing this.  It is not necessary to remove the StarStrings mod or change anything about as v2 will take over where the previous script left off.
